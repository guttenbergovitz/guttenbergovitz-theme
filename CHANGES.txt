JAK UŻYWAĆ NOWYCH NARZĘDZI?
────────────────────────────────────────────────────────────────────────

Walidacja:
  $ make validate         # Sprawdź spójność wszystkich tematów
  $ make analyze          # Szczegółowa analiza kolorów
  $ make help            # Lista wszystkich komend

Wprowadzanie zmian w kolorach:
  1. Edytuj palette.json
  2. make validate (aby zobaczyć co wymaga aktualizacji)
  3. Zaktualizuj wskazane pliki
  4. make validate (dla potwierdzenia)
  5. Commit wszystkich zmian razem
